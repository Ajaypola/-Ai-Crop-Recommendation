# 🌱 Crop Recommendation System

## 📌 Project Overview
The Crop Recommendation System is an AI-based web application that helps farmers and users select the most suitable crop based on environmental and soil conditions.

It uses Machine Learning and data analysis to provide accurate crop suggestions, improving agricultural productivity and decision-making.

---

## 🚀 Features
- 🌾 Crop recommendation based on input data  
- 📊 Machine learning-based prediction  
- 🔐 User authentication (Login & Signup)  
- 🌐 Web-based interface  
- ⚡ Fast and user-friendly  

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Machine Learning: Python (if used)

## ⚙️ Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/Ajaypola/-Ai-Crop-Recommendation
2. Navigate to Project Folder
cd kv-capstone/server
3. Install Dependencies
npm install
4. Setup Environment Variables

Create a .env file and add:

MONGODB_URI=mongodb://127.0.0.1:27017/cropai
JWT_SECRET=your_secret_key
PORT=5000
5. Run the Server
npm run dev
